# Task Manager App

A modern task management app built with React Native and Expo.

## Features

- ✅ Add tasks with descriptions
- ✅ Mark tasks as complete/incomplete
- ✅ Delete tasks with confirmation
- ✅ Edit existing tasks
- ✅ View progress statistics
- ✅ Light/Dark theme support
- ✅ Empty state handling

## Setup Instructions

1. Install dependencies: `npm install`
2. Start the app: `npx expo start`
3. Run on your device using Expo Go or an emulator

## Third-Party Libraries

- **expo-linear-gradient**: Gradient backgrounds and UI elements
- **@expo/vector-icons**: Icon library (Ionicons)
- **expo-crypto**: Generate unique IDs for tasks
- **expo-haptics**: Haptic feedback (if implemented)
- **expo-router**: File-based navigation
- **react-native-safe-area-context**: Safe area handling
